# Rese Wellness — Static Site

Full static site for Rese Wellness, LLC (Gilbert, AZ). No build step — plain HTML/CSS/JS.

## Deploy
- **GitHub Pages:** Settings → Pages → deploy from branch root. `index.html` is the homepage.
- **Netlify:** drag this whole folder onto the deploy area.

## Structure
- `index.html` — homepage (same as "Rese Wellness Site v2.html")
- One .html file per page (programs, assessment, journal, FAQ, legal, contact)
- `assets/` — all images
- To add a journal post, see the how-to comment inside `Journal.html`

## Notes
- Assessment checkout is a design prototype — card fields are not wired to a payment processor.
- Email capture on the hormone waitlist needs a form provider (marked for GHL integration).
